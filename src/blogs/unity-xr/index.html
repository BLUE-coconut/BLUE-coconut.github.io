Unity XR 开发的核心是 OpenXR + XR Interaction Toolkit，本文记录关键流程与踩坑。

环境搭建

推荐配置：

  • Unity 2022.3 LTS 或更新
  • OpenXR Plugin
  • XR Interaction Toolkit (XRI)
  • XR Plugin Management（自动加载各平台 provider）

OpenXR 优势

OpenXR 是跨厂商标准，相比各厂商私有 SDK：

  • 一套代码支持 Quest / Pico / Vision Pro / SteamVR
  • 自动处理输入映射
  • Khronos 维护，长期可靠

手部追踪

Quest 上启用手部追踪：

  1. 在 Oculus Feature Group 勾选 Hand Tracking
  2. XR Hands Package 引入
  3. 绑定 HandVisualizer 预制体

性能优化要点

  • 单眼分辨率建议 1.4x（Quest 2）
  • URP 优于 Built-in（支持 Foveated Rendering）
  • 动态分辨率比固定分辨率体验更好
  • 避免过多数量 Update 调用（XR 设备帧率敏感）
