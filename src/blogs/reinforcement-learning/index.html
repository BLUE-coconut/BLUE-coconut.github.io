强化学习（Reinforcement Learning, RL）是机器学习的第三大范式，核心是"从交互中学习最优策略"。

马尔可夫决策过程（MDP）

RL 的数学基础是 MDP，由五元组 (S, A, P, R, γ) 描述：

  • S：状态空间
  • A：动作空间
  • P：状态转移概率
  • R：奖励函数
  • γ：折扣因子（0-1）

Bellman 方程

状态价值函数的递归定义：

V(s) = E[ R(s,a) + γ * V(s') | s ]

最优价值满足 Bellman 最优方程：

V*(s) = max_a E[ R(s,a) + γ * V*(s') ]

Q-Learning

Q-Learning 是 off-policy TD 控制算法，更新规则：

Q(s,a) ← Q(s,a) + α * [ r + γ * max_a' Q(s',a') - Q(s,a) ]

DQN

DQN（Deep Q-Network）用神经网络近似 Q 函数，两个关键 trick：

  1. Experience Replay：打散时序相关性
  2. Target Network：稳定训练目标

策略梯度

直接参数化策略 πθ(a|s)，用梯度上升最大化期望回报：

∇θ J(θ) = E[ ∇θ log πθ(a|s) * Gt ]

REINFORCE 是最简单的实现，方差大，实际多用 Actor-Critic 框架。

PPO

PPO（Proximal Policy Optimization）是当前最主流的 on-policy 算法，核心是 clipped surrogate objective：

L(θ) = E[ min( rt * At, clip(rt, 1-ε, 1+ε) * At ) ]

PPO 在 ChatGPT 的 RLHF 阶段也扮演关键角色。
