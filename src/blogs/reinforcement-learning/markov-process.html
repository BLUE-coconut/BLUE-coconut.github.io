马尔可夫过程（Markov Process）是强化学习数学建模的起点。 从最简单的马尔可夫链到完整的马尔可夫决策过程（MDP），每一步扩展都为强化学习引入一类新的问题。

一、马尔可夫性

马尔可夫性是指系统的下一状态仅与当前状态有关，与之前的状态无关。 用数学语言表述为： 

P(S_{t+1} | S_t) = P(S_{t+1} | S_1, S_2, ..., S_t)

即：给定当前状态 S_t，下一状态 S_{t+1} 的条件概率与更早的历史 S_1, ..., S_{t-1} 无关。 「现在决定未来，过去不决定未来」——这是马尔可夫性的核心直觉。

具备马尔可夫性的状态称为马尔可夫状态。 在强化学习中，状态是否满足马尔可夫性，直接决定了能否套用 MDP 框架进行求解。

二、马尔可夫过程（Markov Process, MP）

马尔可夫过程（又叫马尔可夫链，Markov Chain）是满足马尔可夫性的随机过程， 由二元组 <S, P> 组成：

  • S：有限状态集合
  • P：状态转移概率矩阵，P(s, s') = P(S_{t+1} = s' | S_t = s)

从某一状态出发，根据状态转移概率生成完整的状态序列：S_1, S_2, ..., S_t, ...，这叫做采样（sampling）。

示例

以「学生一天」为例（无动作、无奖励），状态集合与转移关系：

状态含义
class 1第一节课
class 2第二节课
class 3第三节课
sleep睡觉

所有状态间的转移概率构成一个 4×4 的矩阵 P，从初始状态出发反复采样，可以得到无数条「状态序列」。

三、马尔可夫奖励过程（Markov Reward Process, MRP）

在马尔可夫过程的基础上加入奖励函数 R 和折扣因子 γ，得到四元组 <S, P, R, γ>：

  • R：奖励函数，R(s) = E[R_{t+1} | S_t = s]，表示在某状态下获得的即时奖励的期望
  • γ：折扣因子，γ ∈ [0, 1]，权衡「即时奖励」与「未来奖励」的重要性

回报（Return）

从时刻 t 开始到过程结束的所有奖励的折扣累加和称为回报 G_t： 

G_t = R_{t+1} + γ · R_{t+2} + γ² · R_{t+3} + ...
    = Σ_{k=0}^∞ γ^k · R_{t+k+1}

γ 接近 0：更关注「即时奖励」（短视）
γ 接近 1：更关注「长期奖励」（远视）

价值函数（Value Function）

状态 s 的价值定义为从该状态出发的回报的期望： 

V(s) = E[G_t | S_t = s]

它满足贝尔曼方程（Bellman Equation）：

V(s) = R(s) + γ · Σ_{s'∈S} P(s, s') · V(s')

即：当前状态的价值 = 即时奖励 + 折扣后的下一状态价值的期望。 通过联立所有状态的方程可以求解出 V(s)，但实际中常用迭代法（动态规划 / 蒙特卡洛 / TD 学习）。

四、马尔可夫决策过程（MDP）

在 MRP 的基础上再加入动作 A，就得到了马尔可夫决策过程，由五元组 <S, A, P, R, γ> 组成：

  • S：状态集合
  • A：动作集合
  • P：状态转移概率，P(s'|s, a) = P(S_{t+1}=s' | S_t=s, A_t=a)
  • R：奖励函数，R(s, a) = E[R_{t+1} | S_t=s, A_t=a]
  • γ：折扣因子

MDP 是强化学习最经典、最核心的数学框架。 在 MDP 中，策略 π 决定智能体在不同状态下如何选择动作： 

π(a | s) = P(A_t = a | S_t = s)

给定策略 π 后，MDP 就退化为一个 MRP，因此策略 + MRP = 采样轨迹。

策略下的价值函数

在策略 π 下，状态价值函数和行为价值函数定义为： 

V_π(s) = E_π[G_t | S_t = s]
Q_π(s, a) = E_π[G_t | S_t = s, A_t = a]

两者满足关系：

V_π(s) = Σ_{a∈A} π(a|s) · Q_π(s, a)
Q_π(s, a) = R(s, a) + γ · Σ_{s'∈S} P(s'|s, a) · V_π(s')

贝尔曼最优方程

强化学习的最终目标是找到最优策略 π*，使得价值最大化。最优价值函数满足贝尔曼最优方程： 

V*(s) = max_a [ R(s, a) + γ · Σ_{s'∈S} P(s'|s, a) · V*(s') ]
Q*(s, a) = R(s, a) + γ · Σ_{s'∈S} P(s'|s, a) · max_{a'} Q*(s', a')

一旦解出 V* 或 Q*，最优策略可以立即给出： 

π*(a|s) = 1  if a = argmax_a Q*(s, a)
        0  otherwise

五、概念演进总结

概念组成引入的关键要素
马尔可夫过程（MP）<S, P>状态 + 转移概率
马尔可夫奖励过程（MRP）<S, P, R, γ>+ 奖励函数、折扣因子
马尔可夫决策过程（MDP）<S, A, P, R, γ>+ 动作、策略

可以看到，从 MP 到 MRP 到 MDP 是一个逐步引入决策与价值的过程： MP 描述「环境的随机演化」，MRP 描述「环境演化 + 目标信号」，MDP 描述「环境 + 目标 + 智能体的决策」。 强化学习的所有算法（Q-learning、Policy Gradient、Actor-Critic 等）本质上都是在不知道完整 P 或 R 的情况下，对贝尔曼最优方程做近似求解。

小结

马尔可夫过程为强化学习提供了形式化语言：状态、动作、奖励、折扣因子、策略、价值函数。 所有强化学习算法都是在这些变量的不同组合上做文章—— 有的已知环境模型（Model-Based），有的未知模型只能与环境交互（Model-Free）； 有的直接优化策略（Policy Gradient），有的先学价值再导出策略（Value-Based / Actor-Critic）。

理解 MDP 五元组与贝尔曼最优方程，是理解后续所有 RL 算法的地基。
